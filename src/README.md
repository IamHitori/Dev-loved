# Operadores Aritmeticos

**Los operadores aritmeticos son para poder hacer operaciones matematicas, y los 
que se usan en python son:**

| Signo           | Funcion          | Ejemplo      |
| :-------------: | :-------------:  | :----------: |
| +               | Suma             |     1+2      |
| -               | Resta            |     5-3      |
| *               | Multiplicacion   |     3*4      |
| /               | Divicion         |     4/3      |
| %               | Reciduo          |     3%3      |
| //              | Solo decimal     |   4.5//2.4   |
| -a              | Negacion         |   -3         |
